# Back dashboard salão de beleza

Para rodá-lo, faça:

```console
> npm install ou yarn
> npm run server ou yarn server
```

Ela responderá na porta 5000.


## Rodando o FRONT

[FRONT](https://github.com/angela-oliveira/salon-dashboard)

Para rodá-lo, faça:

```console
> npm install ou yarn
> npm start ou yarn start
```

Ela responderá na porta 3000.
